# 🕌 Masjid Al-Mujahidin - Display Digital

Aplikasi display digital masjid modern yang menampilkan jadwal sholat, jam digital, dan informasi masjid secara real-time.

## ✨ Fitur Utama

- 🕐 **Jam Digital Real-time** - Tampilan waktu yang akurat
- 🕌 **Jadwal Sholat 5 Waktu** - Subuh, Dzuhur, Ashar, Maghrib, Isya
- 🎨 **Desain Modern & Elegan** - Tampilan profesional untuk masjid
- 📱 **Responsive Design** - Cocok untuk berbagai ukuran layar
- 🌅 **Background Masjid** - Visual yang menenangkan dan islami

## 🚀 Cara Menggunakan

### Online (GitHub Pages)
Kunjungi: https://wahyudin-tazkia.github.io/masjid-almujahidin/

### Lokal
1. Download file `index.html`
2. Buka langsung di browser (Chrome, Firefox, Edge, dll)
3. Aplikasi siap digunakan!

## 📱 Penggunaan

- **Display Masjid:** Tampilkan di layar besar di masjid
- **Info Jamaah:** Share URL ke jamaah untuk akses mobile
- **Update Jadwal:** Edit file HTML untuk mengubah jadwal sholat

## 🛠️ Kustomisasi

Untuk mengubah:
- **Nama Masjid:** Edit bagian `<title>` dan nama masjid di HTML
- **Jadwal Sholat:** Update data waktu sholat di script
- **Warna & Tampilan:** Modifikasi CSS di bagian `<style>`
